# @luxtudio/core-bluetooth

Core Bluetooth module for React Native

## Installation

```sh
yarn add @luxtudio/core-bluetooth
```
or
```sh
npm install @luxtudio/core-bluetooth
```

## Usage

```js
import {} from '@luxtudio/core-bluetooth';

// ...

```
